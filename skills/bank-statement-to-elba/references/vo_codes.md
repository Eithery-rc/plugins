# Коды ВО (валютных операций) — Инструкция ЦБ РФ 181-И, Приложение 1

Ключевые коды для ИП с иностранной выручкой на зарубежных счетах.

## 20200 — Зачисление от нерезидента (услуги/работы/ИС)

**Применяется:** оплата иностранного контрагента за твои услуги, инвойсы, SWIFT-поступления. Основной код для фрилансеров/агентов.

**Признаки:**
- `direction=in`
- контрагент — юрлицо-нерезидент (не из РФ)
- назначение содержит: `invoice`, `payment`, `services`, `agent fee`, «оплата услуг»

## 61100 — Перевод резидентом на свой счёт за пределами РФ

**Применяется:** переводы между собственными зарубежными счетами (например, Jusan KZ → TBC GE).

**Признаки:**
- `direction=out`
- контрагент-получатель = сам резидент (по IBAN/имени совпадает)
- назначение: `transfer to own account`, «перевод на свой счёт»

## 80150 — Комиссии банка-нерезидента

**Применяется:** списания за обслуживание счёта, комиссии за переводы, пакеты услуг.

**Признаки:**
- `direction=fee` (или `out` с очевидным «комиссия»)
- получатель = сам банк
- малая сумма (обычно < 100 USD)
- назначение: `fee`, `maintenance`, `commission`, «комиссия», «обслуживание»

## Правила классификации (для AI)

1. Если `direction=in` и контрагент иностранный → **20200**.
2. Если `direction=out` и контрагент — это сам резидент (совпадение по имени / IBAN в `profile` или `contractors`) → **61100**.
3. Если `direction=out` или `fee` и назначение содержит маркеры комиссии → **80150**.
4. **В сомнительном случае — спросить пользователя.** Покажи транзакцию (дата / сумма / направление / контрагент / назначение) и предложи 2–3 наиболее вероятных кода с краткими пояснениями. Прими ответ и используй его. Только если пользователь тоже не может решить — ставь `vo_code="?"` и отметь в `summary.md` как «требуется ручная классификация». Не угадывать.

Аналогично для контрагента: если в PDF имя отсутствует или представлено только IBAN-ом/кодом — спроси пользователя, кто это, и сохрани резолюцию в `~/.config/ip-reports/contractors.json` (следующий прогон будет автоматическим).

Другие коды 181-И существуют (70xxx, 32xxx, etc.), но для MVP покрываем только эти три.
