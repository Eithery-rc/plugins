# Колонки журнала vs Эльбовский xlsx-экспорт

Эльбовский экспорт «Деньги» имеет 19 колонок. Для ПСН-случая релевантны 7 — остальные всегда пустые или константные.

| Колонка (наша) | Колонка Эльбы | Источник |
|---|---|---|
| Дата | Дата | из PDF |
| Контрагент | Контрагент | profile контрагентов |
| Тип документа | Тип документа | константа «Банковский ордер» |
| № документа | № документа | reference из PDF |
| Сумма ₽ (Учитывается в патенте) | Учитывается в патенте | USD × курс ЦБ |
| Тип операции | Тип операции | «Начисление вознаграждения по агентскому договору» |
| Описание | Описание | «поступление средств за {месяц} {год}» |

Добавлены **3 наших колонки** (Эльба их не имеет):

| Колонка | Смысл |
|---|---|
| USD | Исходная сумма в валюте |
| Курс ЦБ | Курс на дату поступления |
| Банк-источник | Jusan / TBC / ... |

Выкинутые эльбовские колонки (всегда пустые / бесполезные для ПСН-импорта):
- Группа контрагента (всегда «Без группы»)
- Тип денег (всегда «Прочее»)
- Поступило / Списано (всегда 0 для ПСН)
- Учитывается в доходе УСН / расходе УСН (не ПСН)
- Платежные реквизиты контрагента / Ваши платежные реквизиты
- Источник появления в системе / Дата появления в системе
- Column18 (валюта «р.»)
